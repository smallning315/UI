貓貓的各種花色

  • 黑貓
  • 白貓
  • 虎斑貓
  • 玳瑁貓

貓咪的叫聲非常多變，有長、短、高、低，也有哈氣聲、低吼聲、呼嚕聲等，簡直是百變歌喉！貓咪會透過聲音，向人類或是同伴表達各種情緒，而當貓咪發出呼嚕聲時，代表他們感到非常舒適與滿足，心情十分愉悅～但毛爸媽要注意，有時貓咪感到疼痛不舒服，或是母貓在生產時，也會發出呼嚕聲來安慰自己，毛爸媽要仔細觀察主子呼嚕呼嚕叫的原因，可別大意了！

貓咪尾巴豎直的動作其實源於年幼時，母貓舔拭幼貓肛門、尿道刺激排泄的記憶。因此成年貓咪高舉尾巴時，代表他們防備心較低，是對毛爸媽展現熱情、濃厚愛意、撒嬌的動作，此時毛爸媽可以多撫摸貓貓、和他們玩遊戲，是個增進情感的好機會呢！此外，貓咪尾巴自然下垂時，也代表他們覺得十分悠閒，毛爸媽也可以多把握機會和他們培養感情喔！
